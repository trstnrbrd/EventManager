# EventHub - Community Event Manager

Welcome to **EventHub**, a community-focused platform designed to simplify event management and help users connect with their local community.  

This project was created as part of a **Data Structures and Algorithms** course project by a 2nd-year BSIT student.

## Live Demo
Check out the live website here: [EventHub Booking Page](https://reboredotristan.pythonanywhere.com/booking/)

## Features

- **Create Events:** Organize and host community events easily, from workshops to meetups and celebrations.  
- **Discover Events:** Browse upcoming events in your area that match your interests.  
- **Book Your Spot:** Reserve a spot for events directly on the platform.  
- **User-Friendly Interface:** Simple design using HTML, CSS, and Bootstrap. No frontend frameworks required.  

## About

EventHub was developed to answer the question posed in our project:  
*"If you want to develop a system or web, what would you do to help the community?"*  

The goal is to foster community spirit by providing an accessible tool for event organizers and attendees to connect and participate.  

## Tech Stack

- **Backend:** Python Django  
- **Frontend:** HTML, CSS, Bootstrap (no frontend frameworks)  
- **Database:** SQLite (default for Django projects)
